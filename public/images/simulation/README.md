# Simulation Scene Images

This directory contains images for simulation scenes. Each scene category has its own subdirectory.

## Directory Structure

```
images/simulation/
├── home/
│   ├── home_1.jpg
│   ├── home_2.jpg
│   ├── home_3.jpg
│   ├── home_4.jpg
│   ├── home_5.jpg
│   └── default.jpg (fallback)
├── street/
│   ├── street_1.jpg
│   ├── street_2.jpg
│   ├── street_3.jpg
│   ├── street_4.jpg
│   └── default.jpg (fallback)
└── upper_floor/
    ├── upper_floor_1.jpg
    ├── upper_floor_2.jpg
    ├── upper_floor_3.jpg
    ├── upper_floor_4.jpg
    └── default.jpg (fallback)
```

## Image Requirements

### Technical Specifications
- **Format**: JPG or PNG (JPG recommended for smaller file size)
- **Aspect Ratio**: 16:9 or 4:3 (landscape orientation)
- **Resolution**: Minimum 1920x1080 (Full HD), recommended 2560x1440 (2K)
- **File Size**: Keep under 500KB per image for fast loading
- **Optimization**: Compress images before adding (use tools like TinyPNG, ImageOptim)

### Content Guidelines
- **Child-Friendly**: No graphic or scary imagery
- **Educational**: Show realistic but safe flood scenarios
- **Clear**: Images should clearly represent the scene situation
- **Appropriate**: Suitable for ages 12-16

## Scene Descriptions

### Home Scenes
- **home_1**: Flooding inside a home (living room with rising water)
- **home_2**: Water rising fast, knee-deep in a room
- **home_3**: Power outage, dark interior scene
- **home_4**: Emergency supplies/kit visible
- **home_5**: Final evacuation decision scene

### Street Scenes
- **street_1**: Flooded street with moving water
- **street_2**: People on street, evacuation scenario
- **street_3**: Vehicles and water on street
- **street_4**: Higher ground or safe area

### Upper Floor Scenes
- **upper_floor_1**: Safe upper floor location
- **upper_floor_2**: Signaling for help from upper floor
- **upper_floor_3**: Waiting for rescue scenario
- **upper_floor_4**: Final safe location

### Important Notes
- Ensure images are royalty-free and can be used commercially
- Check licensing (CC0, CC-BY, or similar)
- Edit images to be child-appropriate if needed
- Add subtle overlays if images are too intense

## Fallback Images

Each category should have a `default.jpg` that represents the general scene type. This will be used if a specific scene image fails to load.

## Testing

After adding images:
1. Test each scene to ensure images load correctly
2. Verify images display properly on different screen sizes
3. Check that fallback works if an image is missing
4. Ensure text overlay is readable over images
